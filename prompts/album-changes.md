# Album Changes

## Album Date

- Add a date field to albums which can be null
- If the album has a valid date, do not allow uploads by guests to the album before that date.
- Admins and event managers should be allowed to upload to an album regardless of date restrictions.

## Hide Album

- Add the ability to hide an album from guests.
- Hidden albums should be visible to Admins and Event Managers in the gallery and have an icon next to their name to show they are hidden.
